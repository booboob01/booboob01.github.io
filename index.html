A complete credit-risk workflow: loan-level data cleaning, feature engineering (debt-service ratios, delinquency flags, vintage segmentation, high-cardinality encoding), Probability-of-Default training with XGBoost, benchmarking against logistic regression under severe class imbalance using stratified k-fold cross-validation, and probability calibration via Platt scaling so outputs can be consumed downstream as proper PD estimates.

Expected loss is simulated at the portfolio level through PD × LGD × EAD, matching the Basel framework. SHAP reasons are produced per prediction, and a model card — version, trained timestamp, metrics, and known failure modes — ships with the bundle for auditability and governance review. Evaluated on ROC AUC and PR AUC rather than raw accuracy, which is the correct call when defaults are rare.

02 · PHYSICS · MULTI-CLASS ML

LIGO Gravitational-Wave Glitch Classifier

Python · scikit-learn
PHYS 310 · 2026

Multi-class classification of 7,966 detector-noise samples across 22 severely imbalanced glitch classes (67× ratio between largest and smallest) from the LIGO Gravity Spy dataset. The problem: separate real astrophysical signal candidates from instrumental artefacts, where naive accuracy would be wildly misleading.

Compared Logistic Regression, SVM (RBF), and Random Forest under class_weight='balanced' with stratified splits. Validated stability with 3-fold cross-validation and tuned the Random Forest via grid search over depth, estimators, and split thresholds. Feature importance placed peak_frequency (27%), amplitude (16%), and SNR (15%) as top predictors — matching physical intuition about how glitches separate in time-frequency space.

04 · EMBEDDED · SIGNAL PROCESSING

Biometric Lie Detector

MSP430F5529 · Python
PHYS 319 · 2026

End-to-end physiological stress-detection system, built from bare metal up. A MAX30102 optical pulse sensor and a DIY galvanic skin response circuit (voltage divider + elastic band stabilisation) stream 100 Hz multi-channel data over UART into a Python signal-processing pipeline.

I²C was implemented in software from scratch after hardware I²C showed bus contention. The Python stack applies Butterworth bandpass (0.5–4 Hz), IIR tonic/phasic GSR decomposition, RMSSD-HRV calculation, and adaptive peak detection with quality-gating on inter-beat-interval variability. Five physiological signals fuse into a research-weighted 0–100 deception score.
