01 / Featured

Banking Risk Platform

End-to-end credit risk scoring platform built in R. Trains and validates a Probability-of-Default (PD) model using XGBoost, explains individual predictions with SHAP values, and simulates portfolio-level expected loss (PD × LGD × EAD) following Basel framework principles.

Includes a Shiny dashboard for both batch CSV scoring and single applicant decisions, with governance built in — model card, version tracking, timestamp, and diagnostics.

02

Biometric Lie Detector

End-to-end embedded lie detection system using an MSP430F5529 microcontroller, MAX30101 optical pulse sensor, and a DIY galvanic skin response circuit built from a voltage divider. Streams RED, IR, and GSR data at 100 samples/sec over UART to a real-time Python signal processing dashboard.

Implemented software bit-bang I²C from scratch after hardware I²C caused persistent bus contention. Applied IIR high-pass filtering, Butterworth bandpass (0.5–4 Hz), RMSSD heart rate variability calculation, and GSR SCL/SCR decomposition — all fused into a weighted 0–100 deception score with a live 7-panel Matplotlib dashboard.

03

LIGO Gravitational Wave Glitch Classifier

Multi-class classification of noise artifacts (glitches) in LIGO gravitational wave detector data using the Gravity Spy dataset — 7,966 samples across 23 imbalanced classes sourced from the Laser Interferometer Gravitational-Wave Observatory.

Compared Logistic Regression, SVM, and Random Forest. Handled class imbalance with stratified splits and class weighting. Random Forest achieved 86% weighted F1. Feature importance analysis identified SNR, peak frequency, and signal duration as the strongest predictors.
